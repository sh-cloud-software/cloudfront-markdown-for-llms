Welcome to CloudFront Markdown for LLMs

This project demonstrates serving Markdown content to LLMs while maintaining HTML pages for human visitors.

How It Works

  1. HTML files are uploaded to an S3 bucket.
  2. A Lambda function automatically converts them to Markdown.
  3. CloudFront checks the Accept header on each request.
  4. If Accept: text/markdown is present, it serves the .md version.
