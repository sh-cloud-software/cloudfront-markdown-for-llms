About This Project

This is a demonstration of content negotiation using CloudFront Functions.

Architecture

The solution uses the following AWS services:

  • Amazon S3 - stores both HTML and generated Markdown files
  • AWS Lambda - converts HTML to Markdown using the turndown library
  • Amazon CloudFront - serves content with header-based routing
  • CloudFront Functions - performs URL rewriting at the edge

Benefits for LLMs

Large Language Models process Markdown more efficiently than HTML because:

  1. Markdown has less syntactic overhead.
  2. The content-to-markup ratio is higher.
  3. Token usage is reduced significantly.
